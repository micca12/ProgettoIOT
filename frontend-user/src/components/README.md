# Components

Componenti React riutilizzabili (.tsx)

Esempi:
- `NFCScanner.tsx` - Componente per scansione NFC
- `QRScanner.tsx` - Componente per scansione QR code
- `LockerCard.tsx` - Card per visualizzare info armadietto
- `Header.tsx` - Header dell'app
- `Button.tsx` - Componenti UI riutilizzabili
