# Pages

Pagine dell'applicazione (.tsx)

Esempi:
- `Login.tsx` - Pagina di login
- `Dashboard.tsx` - Dashboard utente con info armadietto
- `ScanPage.tsx` - Pagina per scansione NFC/QR
- `CheckIn.tsx` - Pagina check-in (assegnazione armadietto)
- `CheckOut.tsx` - Pagina check-out (rilascio armadietto)
